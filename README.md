## Getting started

### Install dependancies

```
yarn
```

### Run application

```
yarn start
```

Go to http://localhost:3000.

### Storybook

We use [Storybook](https://storybook.js.org/).

```
yarn run storybook
```

Go to http://localhost:9009/.
